## Installation

1. Make sure dependencies are installed/updated and ensured before this script.
2. Add `bcc-NEW` folder to your resources folder.
3. Add `ensure bcc-NEW` to your `server.cfg`
4. Run the included database file `NEW.sql`
5. Restart the server.
